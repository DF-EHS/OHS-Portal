系統設定說明
1

初次設定 — 備份附件至本機

K 槽連線後執行：
Copy-Item "K:\HR-人資\7.工安\18.節日天災宣導資料及檢核表\災前及災後檢核表.doc" "C:\Users\gloom.lai\OHS-Portal\dispatch\typhoon\災前及災後檢核表.doc"

2

確認 pywin32 已安裝

pip install pywin32

3

測試 API 回應

python typhoon_dispatch.py --test

4

測試草稿格式

python typhoon_dispatch.py --force

5

啟動工作排程器（以系統管理員執行）

在系統管理員 PowerShell 貼入 XML 指令建立任務 OHS-TyphoonDispatch，每日 06:00–22:00 每 30 分鐘自動偵測。

6

更新附件版本

K 槽版本有修改時，重新執行步驟 1 覆蓋本機備份即可。腳本每次觸發時也會自動嘗試從 K 槽同步。
