# 消防管理系統 建置計畫書

> 大豐環保科技股份有限公司 · 職業安全衛生管理系統
> 建立日期：2026-06-18

---

## 一、系統概述

將 K 槽消防管理資料（9 個資料夾、3,586 個檔案、8.6 GB）
轉化為可管理、可追蹤的 Web 介面。

**核心設計原則**：K 槽文件留在原地，Portal 只建「管理層」——追蹤、提醒、快速存取。

## 三、三大模組規劃

### 模組一：合規儀表板（待建置）

**目標**：一眼看出哪個站的消防演練/檢修申報完成了沒。

**顯示內容**：
- 消防演練（每年6月、12月）× 各站 → 完成/未完成
- 消防設備年度檢修申報 × 各站 → 完成/未完成
- 各站進度 Traffic Light

**資料維護方式**：手動更新 JS 陣列（每年 2-3 次），極低維護成本。

**需要的資料**：
- [ ] 114年、115年各站演練完成日期
- [ ] 114年、115年各站檢修申報完成日期
- [ ] 各站申報書 SharePoint 連結
- [ ] 各站演練成果 SharePoint 連結
