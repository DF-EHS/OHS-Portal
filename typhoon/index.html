系統設定說明
1

初次設定 — 備份附件至本機

K 槽連線後，在 PowerShell 執行以下指令，將檢核表複製到本機（避免 K 槽斷線影響派送）：
Copy-Item "K:\HR-人資\7.工安\18.節日天災宣導資料及檢核表\災前及災後檢核表.doc" "C:\Users\gloom.lai\OHS-Portal\typhoon\災前及災後檢核表.doc"

2

確認 pywin32 已安裝

Outlook COM 物件需要 pywin32，安裝方式：
pip install pywin32

3

測試 API 回應

先測試氣象署 API 是否正常運作（不會建立郵件草稿）：
python typhoon_dispatch.py --test

4

測試草稿格式

強制觸發一次草稿（不受警報狀態限制，用於確認收件人、附件、格式）：
python typhoon_dispatch.py --force

5

啟動工作排程器（以系統管理員執行）

以系統管理員身分在 PowerShell 執行：
.\register_typhoon_task.ps1
完成後可在「工作排程器」中查看任務 OHS-TyphoonDispatch， 每日 06:00–22:00，每 30 分鐘自動檢查一次。

6

更新附件版本

若 K 槽上的檢核表有修改，重新執行步驟 1 的指令覆蓋本機備份即可。 腳本也會在每次有警報時自動嘗試從 K 槽同步。
